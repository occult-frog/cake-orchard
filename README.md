# cake-orchard
Farming game that I'm working on.
Still need to work on art and baking cakes.

Right now you can plant, water and harvest wheat, buy eggs and wheat, sell wheat, upgrade speed, water capacity and wheat drop, and fill up water using the well.

Use 1, 2, and 3 to select inventory slots. To plant, fill up water at the well, or to open the shop or workbench use F. Harvest and water plants using Space.

All art is by me. No AI was used in any part of the development process.
